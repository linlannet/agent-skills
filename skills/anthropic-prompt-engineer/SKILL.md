# Anthropic 提示工程师

掌握Anthropic经过验证的提示工程技术的艺术和科学。使用Claude AI模型（Claude 4.x、Sonnet、Opus、Haiku）的最佳实践从头生成新提示或改进现有提示。

## 此技能的功能

通过Anthropic的官方技术帮助您为Claude AI创建和优化提示：
- **生成新提示** - 根据需求构建有效的提示
- **改进现有提示** - 优化提示以获得更好的结果
- **应用最佳实践** - 使用Anthropic的经过验证的技术
- **避免常见错误** - 防止幻觉和不清晰的输出
- **为Claude 4.x优化** - 利用最新模型功能
- **构建复杂提示** - 构建多步骤、生产就绪的提示

## 提示工程的重要性

**没有适当提示的后果：**
- 不一致或不正确的输出
- 幻觉和虚构信息
- 不清晰或冗长的响应
- 浪费的token和API调用
- 复杂任务性能差
- 难以重现结果

**使用精心设计的提示的好处：**
- 精确、可靠的输出
- 事实、有根据的响应
- 清晰、格式化的结果
- 高效的token使用
- 出色的复杂任务性能
- 可重现、生产就绪的结果

## 核心技术摘要

### 1. 清晰直接
提供明确、无歧义的指令。Claude 4.x在精确指导下表现出色。

### 2. 使用XML标签进行结构组织
使用语义标签如`<instructions>`、`<example>`、`<context>`组织提示。

### 3. 思维链（CoT）
要求Claude为复杂推理逐步思考。

### 4. 预填充
开始Claude的响应以引导格式和风格。

### 5. 少样本示例
提供2-5个多样化的示例，展示您想要的模式。

### 6. 角色分配
给Claude一个特定的角色或身份，以获得适当的上下文。

## 关键原则

1. **经验方法** - 测试、测量、迭代
2. **上下文作为资源** - 每个token都重要
3. **清晰优于聪明** - 明确的指令效果最好
4. **示例教学最佳** - 展示，而不仅仅是告诉
5. **结构有帮助** - 组织减少混淆
6. **迭代改进** - 基于结果完善
